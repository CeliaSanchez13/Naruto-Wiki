Author notes

This page has been created for Angular language training.

The following tools have been used for the content of this page:

  • Firebase The "Realtime Database" functionality has been used for data storage.
  • Firebase (Hosting) We have used the hosting provided by Firebase totally free to deploy the site.
  • Naruto DB All the information of the page has been obtained through the result of the requests made to this API. (Sometimes it is difficult to load some images on some sites)
  • Giphy Developers API used to obtain the GIFs within the web page.

For the design and functionalities, we used Ng-bootstrap , Bootstrap, Font Awesome, Angular Material ,Sweet Alert2 and Animate.css. As main language and all its composition, Angular has been used.

Note: This page has a lot of improvements to make in the future, among them... make the page responsive. It has some points that are pending to be fixed.
